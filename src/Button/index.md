# Button

## 引入

```
import { Button } from 'poplar'
```

## 使用方法

## 基本使用

```
<Button>默认按钮</Button>
```

### 禁止按钮

```
<Button disabled="true">默认禁止</Button>
```

### primary 按钮

```
<Button type="primary">primary 按钮</Button>
```

### ghost 按钮

```
<Button type="ghost">ghost</Button>
```
